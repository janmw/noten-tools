## 📂 Dateistruktur
Ich empfehle die folgende Dateistruktur. So ist gesichert, dass alle Stücke sauber voneiander getrennt gespeichert werden und jederzeit Dateien effektiv gesucht werden können. 

```
├─ 0000 - Inbox
│  ├─ Scan1.pdf
│  └─ Scan2.pdf
│
├─ 0001 - Notensatz 1
│  ├─ 0001 - Sonstige Stimmen
│  │  ├0001 - Notensatz 1 - unbenötigte Stimme1.pdf
│  │  └0001 - Notensatz 1 - unbenötigte Stimme2.pdf
│  │
│  ├─ 0001 - Notensatz 1 - 05 F-Horn1.pdf
│  ├─ 0001 - Notensatz 1 - 05 F-Horn2.pdf
│  ├─ 0001 - Notensatz 1 - 07 Bariton TC.pdf
│  └─ 0001 - Notensatz 1 - 09 Tuba.pdf
│
├─ 0002 - Notensatz 2
│  ├─ 0002 - Sonstige Stimmen
│  │  ├0002 - Notensatz 2 - unbenötigte Stimme1.pdf
│  │  └0002 - Notensatz 2 - unbenötigte Stimme2.pdf
│  │
│  ├─ 0002 - Notensatz 2 - 05 F-Horn1.pdf
│  ├─ 0002 - Notensatz 2 - 05 F-Horn2.pdf
│  ├─ 0002 - Notensatz 2 - 07 Bariton TC.pdf
│  └─ 0002 - Notensatz 2 - 09 Tuba.pdf
```
- Jeder Notensatz hat einen eigenen Ordner - Alle Archivnummern sind 4-stellig
- Jeder Notensatz hat einen Unterordner `[Archivnummer] - Sonstige Stimmen` für selten oder gar nicht gebrauchte Noten (z.B. Es-Horn / Posaune in Bb)
- Es gibt einen Inbox-Ordner (0000 (A-Z)/ 9999 (Z-A); Sollte immer oben sein) in dem alle Scans abgelegt und verarbeitet werden.

## 📝 Nomenklatur
### Ordner
- Bennenung: `[Archivnummer] - [Titel]`
- Jeder Unterordner sollte auch die Archivnummer enthalten

### Dateien
- Benennung: `[Archivnummer] - [Titel] - [Register-Code] [Stimme].pdf`

### Register-Codes

| Nr   | Register                                   |
| ---- | :----------------------------------------- |
| 00   | Partitur, Direktion                        |
| 01   | Flöten, Piccolo                            |
| 02   | Oboe, Englischhorn, Fagott                 |
| 03   | Klarinetten *(inkl. Es-, Alt-, Bass-)*     |
| 04   | Saxophone *(Sopran-, Alt-, Tenor-, Bari-)* |
| 05   | Hörner *(F-, Es-)*                         |
| 06   | Trompeten, Flügeöhörner, Kornette          |
| 07   | Bariton, Tenorhorn, Euphonium              |
| 08   | Posaunen                                   |
| 09   | Bässe, Tuben                               |
| 10   | Schlagwerk                                 |
| 11   | Streicher & Sonstiges                      |
